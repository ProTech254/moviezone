MVC Template - All Movies (Palceholder)

Welcome to the MVC Template.

This web app is a very light weight, barebones template that uses OOP PHP, utilising the MVC design pattern and AJAX. This should be used in conjunction with the bv_caryard app and it is hoped that the minimal design will help you follow how the MVC and AJAX approach works.

Click the Select a Movie button and then select a movie listed in the dropdown box. The dropdown select box allows you to select a single movie and is output using the print_r() php function.
